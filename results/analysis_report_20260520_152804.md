## 代数闭包存在性证明中的选择公理需求

### 1. 核心问题

要构造一个域的代数闭包，我们需要：
- 找到所有不可约多项式
- 为每个不可约多项式选择一个根
- 这个过程可能需要进行"无穷多次选择"

### 2. 具体分析

**构造性方法的局限：**
```
对于有限域 F，我们想构造代数闭包的过程：
1. 找到 F[x] 中所有不可约多项式
2. 对每个不可约多项式 f(x)，需要选择一个根
3. 这个根不在 F 中，需要"创造"出来
4. 重复这个过程无穷多次
```

**问题出现：**
- 当不可约多项式集合是可数无穷时，我们需要对每个多项式都选择一个根
- 这需要同时做出无穷多次选择
- 这正是选择公理发挥作用的地方
